# FileSharePlatform

Local file sharing platform built with ASP.NET Core.

## Project Description

This project is a backend server for a file sharing platform.  
Users will be able to register, log in, upload files and share them with others.

The server is built using ASP.NET Core and follows a layered architecture.

## Technologies Used

- ASP.NET Core
- Entity Framework Core
- SQLite
- C#
- Git / GitHub

## Project Structure

Controllers – API endpoints  
Services – business logic  
Models – database models  
DTOs – data transfer objects  
Data – database context and configuration  
Migrations – database schema migrations

## Current Features

- User model
- Database setup with Entity Framework
- SQLite database
- Authentication service
- User registration endpoint
